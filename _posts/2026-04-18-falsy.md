## falsy
falsy는 정확히 `false`값은 아니지만 JavaScript가 `false`로 취급하는 **거짓같은 값**이다.  
대표적인 예로 다음이 있다.
```TypeScript
undefined
null
0
""
NaN
```

## 함수의 매개변수
JavaScript에서 함수 매개변수가 제공되지 않으면 함수 내부의 인숫값은 `undefined`가 된다.  
선택적 매개변수를 정의하면 항상 뒤에 `|undefined`가 유니언 타입으로 추가되어 있다.  
따라서 함수 호출 시 선택적 매개변수는 제공할 필요가 없으며 선택적 매개변수 값은 `undefined`가 된다.  

```TypeScript
function greet(name: string, message?:string){
  if(message){     
    return '안녕 ${name}, ${message};
  }
  return '안녕 ${name}'; //message 값을 주지 않으면 undefined가 되어 if(undefined) = false
}
```
